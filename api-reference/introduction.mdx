## How to Navigate

Choose your path based on what you need right now:

<Steps>
  <Step title="Looking up a term?">
    Start with the [Glossary](/glossary). Quick definitions with links to comprehensive guides when you need more context.
  </Step>
  <Step title="Researching a service?">
    Go to [Service Categories](/service-categories). Each guide explains what to look for, red flags to avoid, and questions that separate good providers from great ones.
  </Step>
  <Step title="Ready to compare?">
    Download the checklists in each service guide, then compare professionals in the [Hometrics Directory](https://hometrics.com) based on verified reviews.
  </Step>
</Steps>
